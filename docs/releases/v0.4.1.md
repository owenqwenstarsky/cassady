## Cassady v0.4.1

Cassady v0.4.1 fixes the desktop launch path so `cass desktop` no longer opens a blank white window when an assetless or stale `cassady-desktop` binary is found.

### Downloads

- macOS Apple Silicon: `cassady-v0.4.1-aarch64-apple-darwin.tar.gz`
- Linux x86_64: `cassady-v0.4.1-x86_64-unknown-linux-gnu.tar.gz`
- Linux ARM64: `cassady-v0.4.1-aarch64-unknown-linux-gnu.tar.gz`
- Windows x86_64: `cassady-v0.4.1-x86_64-pc-windows-gnu.zip`

The macOS and Linux archives contain `cass`, `cassady`, and `cassady-desktop`; launch the desktop app with `cass desktop`. The Windows archive contains `cass` and `cassady`. SHA-256 checksum files are included for every archive.

### Highlights

- Rejects stale or incorrectly built `cassady-desktop` binaries that do not contain the bundled frontend assets, replacing a blank window with an actionable error.
- Fails release desktop builds early unless they are built through the Tauri release path with embedded frontend assets.
- Updates desktop build and install documentation to make the required release-built companion binary explicit.

### Upgrade notes

No config migration is required. If `cass desktop` previously opened a blank white window, update both `cass` and the companion `cassady-desktop` binary from the v0.4.1 macOS/Linux archive, or reinstall the npm package once v0.4.1 is published.

### Install from source

```sh
cargo install --git https://github.com/owenqwenstarsky/cassady --tag v0.4.1
```

### Verification

Built and tested with the GitHub Actions release workflow:

```sh
cargo test --locked --all-targets
cargo build -p cassady --release --locked --target aarch64-apple-darwin --bin cass --bin cassady
(cd cassady-desktop && cargo tauri build --target aarch64-apple-darwin --no-bundle -- --locked)
cargo build -p cassady --release --locked --target x86_64-unknown-linux-gnu --bin cass --bin cassady
(cd cassady-desktop && cargo tauri build --target x86_64-unknown-linux-gnu --no-bundle -- --locked)
cargo build -p cassady --release --locked --target aarch64-unknown-linux-gnu --bin cass --bin cassady
(cd cassady-desktop && cargo tauri build --target aarch64-unknown-linux-gnu --no-bundle -- --locked)
cargo zigbuild -p cassady --release --locked --target x86_64-pc-windows-gnu --bin cass --bin cassady
```
